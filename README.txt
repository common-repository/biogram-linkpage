A customizable link in bio landingspage specially made for people who want to share more than 1 link in their Instagram Bio.

== Description ==

You want to show multiple links in your Instagram Bio? Biogram Linkpage provides the perfect solution. A customizable landingspage you can add to your Instagram Bio. Just add the Biogram Linkpage template to the landingspage, add your logo and background and start adding links to the page.

Once you're done you can add the URL to your Instagram Bio.

== Installation ==

1. Install the plugin on your WordPress website.
2. Activate the plugin.
3. Go to Pages -> Biogram Linkpage.
4. Change the Page Template to Biogram Linkpage. (right side of the screen)
5. That's all! Start customize the page and add the links you want to share.
